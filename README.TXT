# Personal Portfolio Website

## Author
Blaise Leteipa

## Description
This is my personal portfolio website built as part of the assignment.  
It is a multi-page website that includes:
- Home page (introduction and hero section)
- About page (profile details)
- Portfolio page (projects)
- Contact page (contact form and social links)

## Features
- Responsive navigation bar with mobile toggle
- Contact form with validation
- Dynamic footer year (updates automatically with JavaScript)
- Futuristic styling applied via CSS
- Interactivity with JavaScript (menu toggle, form validation, and custom message on Home)

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Run
1. Download the project folder.
2. Open `index.html` in a web browser.
3. Navigate using the navbar to view other sections.
